## Списки

чтобы добавить ненумированные списки, необходимо выделить пункты звездочкой(*).
например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

чтобы добавить нумерованые списки, необходимо пункты просто пронумеровать.
например вот так 
1. первый пункт
2. второй пункт
